0. 打開終端機（terminal, cmder, iterm2）
1. 移動到資料夾
$ cd 專案資料夾名稱
2. 初始化 git (第一次才需要)
$ git init 
3. 加入 github 遠端位置
$ git remote add origin 你的 github repo 位置

4. 將檔案加入 git track
$ git add .
4. 將檔案 commit 進 git 版本庫
$ git commit -a -m "你的 commit 訊息"
5. 上傳到 github !
$ git push -u origin master
6. 分享連結到作業區！
